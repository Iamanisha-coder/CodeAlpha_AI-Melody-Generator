# 🎵 MelodyAI – AI Music Generator

An AI-powered music generation application built using **Python, Streamlit, TensorFlow, and Music21**. It learns patterns from MIDI files using an LSTM neural network and generates original melodies.

## ✨ Features

- 🎼 AI-powered music generation
- 🎹 Multiple music genres
- 🎨 Dynamic genre-based themes
- 🌌 Modern glassmorphism UI
- 🎵 MIDI generation & playback
- 📥 Download generated music
- 📊 Music statistics dashboard
- 🎧 Interactive music player
- 📱 Responsive design

---

## 🛠 Tech Stack

- Python
- Streamlit
- TensorFlow (LSTM)
- Music21
- PrettyMIDI
- NumPy
